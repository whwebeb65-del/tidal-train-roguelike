# 《最后一班：潮汐列车》玩家功能路线图

## 排序原则

先解决玩家“今天为什么回来、回来后做什么、长期收集什么”，再扩展社交和赛季商业化。每项功能必须有独立开关、埋点和回滚方案；未验证留存前不堆叠更多货币、排行榜和付费入口。

平台边界以抖音开放平台现行规范为准：不使用分享、关注或下载换奖励，不用广告失败阻断主流程，不做现金奖励，不隐藏付费随机概率；上线 UGC 前必须具备审核、过滤、举报和处置能力。

## 已完成：收益基础设施 v1

- 已上线原型级确定性商店：¥6 固定 60 星票、¥18 固定非战力涂装，均为一次性商品。
- 已完成 v2 存档迁移、交易幂等、商品拥有记录、支付漏斗埋点和 Cocos 平台事件桥接。
- 已完成四个自愿广告点位：复活、技能刷新、普通奖励重选、重复普通通关加倍；取消或失败均不消耗机会。
- 已移除分享复活，并关闭每日试炼的广告重选与结算加倍；普通互动不再产出星票。
- 当前仅为本地 Mock。正式支付必须经平台支付和服务端验单，个人收款码不进入客户端。

下一项仍是“新玩家成长手册”：先验证首局完成率、D1 留存和三货币理解度，再继续扩大商业化。

## P1：新玩家成长手册

**玩家价值**

把“第一次升级车站、第一次完成构筑、第一次首通、第一次加入军团”等目标串成 5—7 天成长路线。玩家每次只看到 2—3 个当前目标，减少主页系统过多造成的迷失。

**目标指标**

- 首局完成率提升 5 个百分点。
- D1 留存提升 3 个百分点。
- 新玩家在首日理解三货币用途的比例达到 70%。

**实现成本**

中。需要服务端任务进度、事件回补、幂等领奖和按账号版本化；前端可复用现有里程碑卡片。

**商业化适配**

成长手册本身免费。完成第三日后再展示确定性新手外观包，不把付费设为任务条件，不出售直接跳过核心教学的战力。

**合规边界**

不把分享、关注、看广告或充值设为必做任务；广告只能是可跳过的额外选择，并提供失败兜底。

## P1：乘客/模块图鉴与套装收集

**玩家价值**

给局外养成一个清晰长期目标：查看已发现乘客、模块、词条、来源和搭配建议。收集进度解锁车厢涂装、头像框、档案文本等非战力奖励；套装只提供构筑提示，不要求抽齐才能通关。

**目标指标**

- D7 留存提升 2 个百分点。
- 图鉴周访问率达到 35%。
- 玩家平均尝试构筑种类提升 20%。

**实现成本**

中。需要发现记录、内容目录版本、套装规则、预览资源和缺失来源跳转。

**商业化适配**

可销售确定性的图鉴主题皮肤、车厢陈列位和外观套装；不出售付费随机图鉴条目，不用图鉴百分比制造虚假稀缺。

**合规边界**

付费内容在购买前明确展示物品、数量和价格；若未来加入随机内容，必须独立完成概率公示和平台审核，当前版本不做。

## P2：异步军团 Boss

**玩家价值**

军团成员每天可挑战 1—2 次同一潮汐巨兽，个人伤害累积为全团进度。Boss 分阶段改变规则，击破后全员按参与和团队里程碑领取固定奖励，不要求成员同时在线。

**目标指标**

- 加入军团玩家的 D7 留存提升 3 个百分点。
- 军团周参与率达到 25%。
- 参与者人均每周回访 3 天以上。

**实现成本**

高。必须有服务端战斗摘要校验、次数、Boss 血量原子扣减、阶段切换、赛季结算、反作弊和补偿机制。

**商业化适配**

销售军团旗帜、Boss 击破纪念外观和展示动作；不卖无限挑战次数，不让付费玩家垄断团队奖励。

**合规边界**

不发现金、红包或可提现奖励；排行榜只展示合法昵称与头像，并提供举报和屏蔽能力。

## P2：好友助战与幽灵列车

**玩家价值**

玩家可以挑战好友留下的同种子“幽灵列车”路线，看到其关键构筑选择和结算成绩；好友助战提供一次可预期的小型局内效果，不要求实时组队。

**目标指标**

- 好友挑战卡打开后开局转化率达到 25%。
- 社交来源玩家的次日回访率提升 3 个百分点。
- 同种子复玩率达到 15%。

**实现成本**

中高。需要服务端种子、版本、战斗摘要、好友关系/邀请归因、成绩校验和过期策略。

**商业化适配**

可销售挑战卡样式、幽灵列车轨迹和胜利展示；分享行为本身不发货币，回流奖励由可核验的新玩家/回流玩家完成有效游戏行为后结算，并在上线前再次走平台审核。

**合规边界**

分享必须由玩家主动发起，可取消且不影响主流程；不以“分享成功”作为复活、签到或领奖条件。

## P2：主题赛季航线与确定性通行证

**玩家价值**

每 28 天轮换一条主题航线、三种词缀、一个 Boss 和一组叙事档案。免费路线提供完整玩法内容，付费路线只增加明确展示的外观、纪念徽章和便利性资源。

**前置门槛与目标指标**

- 只有基线 D1 留存达到 35%、D7 达到 12%，并完成至少两轮稳定内容更新后才启动。
- 赛季任务周完成率目标 30%。
- 付费转化关注健康增量，同时监控付费/非付费通关率差距不超过预设警戒线。

**实现成本**

高。需要内容热更新、赛季版本、任务回补、购买恢复、跨日/跨赛季结算和客服补单。

**商业化适配**

适合作为主商业化载体，但所有奖励必须在购买前完整展示；不放付费随机核心战力，不通过过密弹窗催促购买。

**合规边界**

价格、内容、有效期和领取条件清晰展示；未成年人保护、支付、退款和广告频控按正式接入规范执行。

## P3：创作者挑战种子

**玩家价值**

创作者可选择官方词缀、固定种子和挑战条件生成关卡卡片，玩家一键进入相同规则并比较完成方式。创作者只能组合官方安全内容，不能上传脚本、图片或任意关卡代码。

**目标指标**

- 挑战卡到开局转化率达到 20%。
- 创作者挑战带来的新增玩家中，完成首局比例达到 55%。
- 举报率、违规率和低质量重复内容保持在运营阈值内。

**实现成本**

高。需要创作者准入、词库过滤、内容审核、举报、下架、版本兼容、风控和推荐质量控制。

**商业化适配**

可提供官方主题模板和创作者展示外观；不提供现金悬赏或可提现排名奖励，不让付费决定推荐曝光。

**合规边界**

在审核、过滤、举报和运营处置链路完成前不开放用户自定义标题或文本；所有挑战条件由服务器白名单校验。

## 推荐实施顺序

1. 当前版本：累计七日值班簿、收益基础设施 v1。
2. 下一版本：新玩家成长手册。
3. 随后：图鉴与套装收集。
4. 服务端社交稳定后：异步军团 Boss、好友幽灵列车。
5. 留存达标且内容产能稳定后：主题赛季与通行证。
6. 审核和风控成熟后：创作者挑战种子。

每个阶段先灰度 5%—10% 玩家，至少观察一个完整留存周期，再决定扩量、调奖或回滚。签到奖励、成长任务和赛季任务要合并计算经济产出，避免齿轮通胀和星票贬值。
