# project-002-tidal-train-roguelike

## 项目目标

设计并验证一款适合抖音小游戏生态的轻量肉鸽产品，工作名为《最后一班：潮汐列车》。产品重点是短局、单手操作、乘客组合构筑、可持续内容更新和合规商业化。

- 商业化状态：已接入广告、商店、分享的 Mock 边界；正式抖音支付、广告和登录仍需在真机平台接入并做合规审核

## 使用与验证

在项目目录执行：

```powershell
npm install
npm run dev
```

浏览器打开 `http://127.0.0.1:4173/` 即可试玩。可以在“车站”中申请内测资格、领取开服礼包、使用公开测试码 `TIDE2026`、加入潮汐灯塔团、选择两名异步支援队友、查看三种货币、升级车站和地图开放；进入同一局后可重复点击互动点领取分段奖励，连续开炮积累潮汐动能，再用技能、维修和潮汐爆发完成战斗。结算会累计军团远征贡献，达到里程碑后可领取确定性奖励。

常用校验命令：

```powershell
npm test
npm run typecheck
npm run build
```

试玩脚本见 [`docs/testing/prototype-playtest-script.md`](docs/testing/prototype-playtest-script.md)，结果记录模板见 [`docs/testing/prototype-results-template.md`](docs/testing/prototype-results-template.md)。

## 目录说明

- `src/`：可跨引擎复用的规则层、存档、平台 Mock、埋点
- `assets/scripts/`：面向 Cocos Creator 的场景、战斗、奖励、车站和教程脚本骨架
- `web/`：无需安装 Cocos 即可运行的浏览器 MVP
- `docs/`：需求、设计、计划和验证文档

## 本轮战斗与性能增强

- 战斗动作统一由 `src/domain/combat/CombatLoopSystem.ts` 结算：普通攻击、汽笛共鸣、维修车厢、潮汐爆发；连击和动能让重复点击形成短期决策，而不是单纯点击伤害。
- 每个节点确定性生成“平静水域 / 急流冲刺 / 回声浓雾”之一，改变维修、普通攻击或技能的收益，便于玩家复盘和持续更新。
- Web 画面保持事件驱动：状态变化时重绘，不运行持续的业务 `requestAnimationFrame` 循环；战斗板使用 CSS 2.5D、绘制隔离和移动端动画降级。
- 当前版本已加入轻量性能保护，但没有在真实抖音机型上宣称稳定 60 FPS；正式上线前仍需在目标机型上用 Cocos 真机包做帧率、内存、广告回调和支付审核。

## 异步组队与军团远征

- 车站新增“联合作战中心”，原型可加入固定军团“潮汐灯塔团”。
- 列车队最多选择两名支援：领航员提供开场动能，炮术员增加行动伤害，工程师增加最大生命；队伍选择在新战斗节点生效。
- 每局胜利或失败会按结果和完成节点数累计周远征贡献，同一个 `runId` 不会重复贡献。
- 20、50、100 贡献对应三档一次性里程碑，奖励仍使用齿轮、航线徽记和星票，不增加第四种钱包货币。
- 分享招募卡只用于传播和召集，不因完成分享直接发货币，避免把分享变成强制收益入口。
- 当前 Web 版本是本地异步社交原型；真实玩家身份、成员列表、邀请归因、军团 Boss、排行榜、贡献和领奖必须迁移到服务端权威存储后才能正式运营。

## 内测资格、开服礼与礼包码

- 车站新增“首班预约中心”，展示限量 3,000 席内测资格，但不会伪造实时剩余人数；本地原型点击申请后立即模拟获得单账号资格。
- 内测玩家可领取一次先行者补给：60 齿轮、2 航线徽记、1 星票与非战力徽章“潮汐先行者”。
- 开服玩家可领取一次开服列车长礼：188 齿轮、6 航线徽记、3 星票与非战力徽章“开服列车长”。
- 礼包码会自动忽略首尾空格和大小写；公开测试码 `TIDE2026` 提供 66 齿轮和 2 航线徽记，同账号不能重复兑换。
- 活动状态单独保存在 `tidal-train-launch-campaign-v1`，刷新后保留资格、已领礼包、徽章和核销记录；“清空本地存档”会同时清除该状态。
- 正式服必须由服务端事务校验账号、服务器时间、全服名额、码库存、渠道和领取记录，并同时写入资产流水；客户端目录和本地存档不具备防作弊能力。

## 工具链与替代编辑方式

- 常用验证命令：`npm test`、`npm run typecheck`、`npm run build`。
- Web MVP 使用广告、分享和商店 Mock，可直接验证复活、技能刷新、三货币和结算闭环。

- 不使用来源不明的安装包或修改版编辑器；正式广告、分享、登录、支付和合规审核仍需接入抖音小游戏真实 SDK。
