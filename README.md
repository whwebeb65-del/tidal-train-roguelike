# project-002-tidal-train-roguelike

## 项目目标

设计并验证一款适合抖音小游戏生态的轻量肉鸽产品，工作名为《最后一班：潮汐列车》。产品重点是短局、单手操作、乘客组合构筑、可持续内容更新和合规商业化。

## 当前状态

- 阶段：原型实现计划评审
- 日期：2026-07-15
- 当前结论：已确认采用“潮汐列车”方向；尚未开始实现或买量
- 下一步：用户选择执行方式后开始原型实现

## 目录说明

- `docs/`：需求、设计和决策记录
- `docs/superpowers/specs/`：经用户审阅的产品设计规格
- `docs/superpowers/plans/`：按任务拆解的原型实现计划

## 使用与验证

当前阶段不运行游戏代码。实现计划已按规则层、平台适配层、Cocos表现层和测试层拆分；原型阶段再补充可玩性测试、设备兼容性和平台能力验证。
